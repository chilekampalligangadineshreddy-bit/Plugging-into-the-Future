# ⚡ Electricity Consumption Analysis in India (2019–2020)

## 📌 Project Overview

This project analyzes electricity consumption patterns across Indian states from **January 2019 to December 2020**, with a special focus on the impact of the **COVID-19 pandemic** and the nationwide lockdown. The analysis explores overall consumption trends, regional demand variations, and the recovery of electricity usage after lockdown restrictions were eased.

The project is implemented in **Python** using **Pandas**, **NumPy**, and **Matplotlib** for data processing and visualization.

---

## 🎯 Objectives

* Analyze overall electricity consumption trends from 2019 to 2020.
* Compare electricity demand across different regions of India.
* Evaluate the impact of the COVID-19 lockdown on electricity consumption.
* Analyze the recovery of electricity demand after lockdown restrictions.
* Generate charts and summary tables for better understanding of electricity usage patterns.
